Tipos de datos primitivos en Java

Tipo int

Es un tipo de datos numérico de 32 bits, entero con signo
Su rango va desde -2e31 hasta 2e31 - 1

Tipo Float

Es un tipo de datos para almacenar números decimales con precisión de 64 bits
